# cntPoolUsage
## Descripción
cntPoolUsage es un script escrito en python que descarga el pdf que indica uso de las piscina del **Club Natació Terrassa (CNT)** durante el fin de semana.
El script accede a la página web del CNT para descargar el fichero por lo que habría que ejecutarlo el jueves o viernes para obtener los datos de ese fin de semana, puesto que es el día que suelen subir el fichero a la web.
## Ejecución
Doble click en el fichero llamado cntPoolUsage.py o ejecutar el siguiente comando:
```sh
$ python cntPoolUsage.py
```
Esto descargará el fichero pdf que indica el uso de las piscinas con el nombre *Disponibilitat_piscines.pdf*.
